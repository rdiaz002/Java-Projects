#Addition Graphics Objects
This Projects adds 3 additional classes to the Graphics Library. 

##Shape- The base class for the other two. Creates a Shape object that contains (x,y) coordiante and base Color. 

##Triangle -Creates a Triangle object that can be drawn onto a JPanel. 
Contains Methods to Retrive area, perimeter, and also the angles and side lengths of the shape


##Circle-Creates a Circle Object that can be drawn onto a Jpanel. Also includes methods to retrieve Area and Perimeter(Circumference).


Note** These objects can only be drawn onto a Jpanel when utilizing the paintComponent method with Graphics object parameter.**
